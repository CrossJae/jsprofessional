# 第七章 函数表达式

## 关键词：

* 函数声明
    * fuction前面字符都没有
    ```
    function fuctionName(arg0,arg1,arg2){
        // 函数体
    }
    ```
    * 函数声明提升：在读取代码之前会先读取函数声明
* 函数表达式
    * fuction前面有其他字符
    ```
    alert(functionName.name)
    ```
    * 匿名函数
    ```
    var functionName = function(arg0,arg1,arg2){
        // 函数体
    }
    ```
    * 即时函数
    ```
    (function(){
        // 函数体
    })()
    ```
